Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all modifications in the current working branch to the staging area so they are added in the next commit
  • git commit -m "": adds commit message that describes what the commit contains
  • git pull (branch name): receives the branch's modifications into the local environment
  • git push (branch name): pushes the locally made changes to the remote GitHub branch
